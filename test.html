Character Skins

From Granblue Fantasy Wiki
Jump to navigation Jump to search

A character skin (officially called outfit with English-language setting) is an alternate artwork or outfit that you can use to customize your character's appearance. Skins have no effect on a character's ability in battle. In most cases, purchasing a skin does not grant you the actual character to use.

Skins can be used on any version of that specific character. For example, Vira's outfits can be used on her SR version, SSR version, or Summer version. Some skins will change your Home Page's background music if it is displayed as your main artwork using the "Switch" button.

For an in-depth explanation on how to use skins, see Outfits.
