Explanations

  • A valid ID looks like 3040000000.
  • Currently supported: Characters, Skins, Weapons, Summons, NPCs, Enemies and Classes.
  • For Characters and Outfits, add 'n' before the ID to access the NPC files. Example: n3040036000 for Seofon's NPC files.
  • For Enemies, add 'e' before the ID. Example: e6205703 for School Belial.
  • For Classes, put the pair of the class ID and its main proficiency, separated by an underscore: 450301_sw for Relic Buster.
  • Loading can take time, especially for Skins and Multi-Unit Characters, please be patient!
  • This page is static and doesn't host any images, they are all pulled directly from GBF.
  • You can find IDs on a corresponding Wiki page, under the Extra informations.
  • Click on images to access their highest quality.
  • Sky Compass arts have a cyan border.
  • You can bookmark the page after loading an ID to save it.
